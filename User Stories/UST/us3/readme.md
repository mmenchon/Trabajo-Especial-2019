## User Story 3
1.	Description
	- Como usuario quiero registrar el acopio de cada residuo en mi hogar para añadir un producto a la base de datos, y cálculo de volumen del material reciclado.

2.	Acceptance Criteria
	- Verificar si el usuario está logueado.
	- Habilidad de registrar los datos que ingresa el usuario.
	- Verificar si el producto que se desea agregar este registrado,  si no lo está no se recicla.
	- Calcular el volumen del material a reciclar.
